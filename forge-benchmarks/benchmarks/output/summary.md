## Overview

This directory contains the canonical benchmark output for the LLM-Forge project.
Run the benchmarks to populate this file with actual results.

## Running Benchmarks

```bash
# From the forge-benchmarks directory
cargo run --bin run_benchmarks

# Or with verbose output
RUST_LOG=info cargo run --bin run_benchmarks -- -v
```

## Output Structure

- `results.json` - Combined benchmark results in JSON format
- `summary.md` - This file, updated with each benchmark run
- `raw/` - Individual benchmark result files
- `history.jsonl` - Historical results for trend analysis
